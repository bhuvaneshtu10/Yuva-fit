# Yuva-fit
YuvaFit is a personalized fitness and wellness platform that helps users track their health metrics and receive customized diet plans, workout routines, and sleep recommendations. It also features an AI-powered assistant for instant health guidance.
# Features
✔ User Authentication

Secure login & signup using Firebase Authentication

Stores user details (age, height, weight, goal, preferences)

✔ Dashboard with Health Stats

Calculates BMI

Calculates daily calorie needs

Displays basic health insights

Clean UI layout

✔ Personalized Diet Plan

Auto-generates diet plans based on:

Weight loss

Weight gain

Maintenance

✔ Personalized Workout Plan

Generates workouts based on user’s available time

Includes cardio, strength, HIIT, cool-down sequences

✔ Sleep Schedule Generator

Suggests ideal bedtime & wake-up time

Simple science-based sleep cycle recommendation

✔ AI Health Assistant (Optional)

Integrated OpenAI API

Helps users with:

Health questions

Fitness advice

Diet recommendations

Motivation & tips
